# Contributing to Stitch Photo to PDF

Thank you for your interest in contributing to Stitch Photo to PDF! We welcome contributions from everyone.

## Code of Conduct

Please be respectful and considerate of others when contributing to this project.

## How to Contribute

1.  **Fork the repository**: Click the "Fork" button at the top right of the repository page.
2.  **Clone your fork**:
    ```bash
    git clone https://github.com/mazenS1/stitch-photo-to-pdf.git
    cd stitch-photo-to-pdf
    ```
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Create a branch**:
    ```bash
    git checkout -b feature/your-feature-name
    ```
5.  **Make your changes**: Implement your feature or fix.
6.  **Run the app**:
    ```bash
    npm run dev
    ```
7.  **Commit your changes**:
    ```bash
    git commit -m "Description of your changes"
    ```
8.  **Push to your fork**:
    ```bash
    git push origin feature/your-feature-name
    ```
9.  **Create a Pull Request**: Go to the original repository and click "New Pull Request".

## Reporting Issues

If you find a bug or have a feature request, please open an issue in the [Issues](https://github.com/mazenS1/stitch-photo-to-pdf/issues) section.

## Style Guide

- We use **ESLint** and **Prettier** (via standard tooling) for code formatting.
- Please ensure your code passes linting before submitting a PR.
  ```bash
  npm run lint
  ```

Thank you for contributing!
